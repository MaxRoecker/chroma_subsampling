# Chroma Subsampling

A small project to test chroma subsampling methods.


## Dependencies

* [Python](https://www.python.org/) (2 or 3);
* [NumPy](http://www.numpy.org/) (>=1.12.1);
* [OpenCV](http://opencv.org/) (>=2.4);


## Run tests

To run the tests:

```sh
make test__chroma_subsampling
```
